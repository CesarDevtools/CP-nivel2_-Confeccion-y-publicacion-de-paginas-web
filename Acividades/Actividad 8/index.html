HTML

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Se utiliza para estructurar el contenido en la web, definiendo elementos como encabezados, párrafos, enlaces, imágenes y otros componentes.

Conceptos Básicos

HTML es un lenguaje de marcado que utiliza etiquetas para definir la estructura y el contenido de una página web. Las etiquetas HTML son elementos que indican al navegador cómo debe mostrar el contenido. Por ejemplo, la etiqueta <p> se utiliza para definir un párrafo, mientras que la etiqueta <a> se utiliza para crear enlaces.

HTML es fundamental para el desarrollo web, ya que proporciona la base sobre la cual se construyen las páginas web. Junto con CSS (Cascading Style Sheets) y JavaScript, forma el trío básico de tecnologías web.

Estas son algunas de las etiquetas HTML más comunes:

  • <h1>: Encabezado principal
  • <p>: Párrafo
  • <a>: Enlace

Listas

En HTML, puedes crear listas ordenadas y no ordenadas. Las listas son útiles para presentar información de manera estructurada.

Aquí tienes un ejemplo de una lista no ordenada:

  • Elemento 1
  • Elemento 2
  • Elemento 3

Y aquí tienes un ejemplo de una lista ordenada:

  1. Primer elemento
  2. Segundo elemento
  3. Tercer elemento
  1. Listas no ordenadas:
    • Se crean con la etiqueta <ul>.
    • Los elementos de la lista se definen con <li>.
    • No tienen un orden específico.
  2. Listas ordenadas:
    • Se crean con la etiqueta <ol>.
    • Los elementos de la lista se definen con <li>.
    • Los elementos están numerados o tienen un orden específico.

Estructura Semántica

Las etiquetas semánticas en HTML5, como header, main y footer, se utilizan para estructurar el contenido de una página web de manera más clara y comprensible tanto para los desarrolladores como para los navegadores y motores de búsqueda. Estas etiquetas tienen un significado específico que describe el propósito del contenido que contienen.

  • <header>: Representa la cabecera de una página o sección. Generalmente incluye el título, logotipos o menús de navegación.
  • <main>: Contiene el contenido principal de la página. Solo debe haber un `
    ` por página y no debe incluir elementos repetitivos como encabezados o pies de página.
  • <footer>: Representa el pie de página de una página o sección. Suele incluir información como derechos de autor, enlaces a políticas de privacidad o contacto.
